# MCU-Bluetooth-car
  基于蓝牙控制的51单片机小车
## 所使用材料
  - 普中HC6800-ES V2.0 开发板(板载有红外线)（STC89C52）
  - HC06蓝牙模块
  - L298N电机驱动模块
  - 直流电机4个
  - 小车及各种配件
  - 12V电源
  - 手机以及蓝牙串口程序 （Arduino bluetooth controller）
